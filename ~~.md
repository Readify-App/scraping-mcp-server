`git clone` 時に表示された `xcode-select` のダイアログは、コマンドライン開発ツールが未インストールであることが原因です。macOS では git を含む CLI ツールを Apple の Command Line Tools (CLT) から提供しているため、最初の `git` 実行時にインストールを求めるポップアップが出ます。

対処手順:
- 画面に出たインストールダイアログで「インストール」を選択する
  - もしくは `xcode-select --install` を実行
- インストール完了後に再度 `curl ... install.sh | bash` を実行する

CLT が入れば `git clone` が成功し、続きのステップへ進めます。
